# Exemplo 3 - Base de Arvore Binaria de Busca

Este exemplo mostra como uma arvore binaria de busca pode organizar dados de forma mais eficiente para pesquisa.

A ideia e seguir o mesmo estilo didatico do exemplo 2:

- classe de no separada;
- classe base da arvore;
- comentarios claros e linha a linha;
- montagem gradual no `main`;
- busca em ABB mostrando a eficiencia pela quantidade de visitas.

## Como usar este exemplo

1. Leia o arquivo `NoBinario.java` para entender o objeto no.
2. Leia a classe base `ArvoreBinariaABBDidatica.java` para entender a regra da ABB.
3. Execute a demonstracao em `ArvoreBinariaABBDemo.java`.
4. Observe quantos nos sao visitados na busca.

## O que este exemplo ensina

- o que e arvore binaria;
- o que e esquerda e direita;
- como a ABB decide para qual lado ir;
- por que a busca pode ser mais rapida;
- como o numero de comparacoes diminui quando a arvore esta organizada.
