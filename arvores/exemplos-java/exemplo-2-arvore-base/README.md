# Exemplo 2 - Base de Arvore Didatica

Este exemplo foi criado para ser a base mais simples possivel para discutir arvores em sala.

A ideia aqui e começar do zero e explicar, com calma, cada parte da estrutura:

- como criar um objeto NoArvore;
- o que e a raiz;
- o que e um no;
- o que e uma aresta;
- o que e uma folha;
- como a arvore cresce por etapas.

## Como usar este exemplo

1. Leia o arquivo Java junto com os comentarios.
2. Observe a classe abstrata, que concentra as regras comuns da arvore.
3. Veja como o objeto NoArvore e criado com `new` antes de ser ligado a outros nos.
4. Compare a arvore generica com a futura arvore binaria.

## O que vem depois

Este exemplo e a base para os proximos passos didaticos:

- arvore binaria;
- busca em arvore;
- percursos em largura e profundidade;
- remocao de nos.

O objetivo e fazer a evolucao de forma cadenciada, sem pular etapas.
