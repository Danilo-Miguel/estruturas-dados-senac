# Arvores

Esta pasta apresenta os conceitos basicos de arvores de forma simples e gradual.

## Objetivo

O objetivo e entender como uma arvore organiza dados em niveis e como identificar, sem pressa, os nomes mais importantes da estrutura.

## Conceitos abordados

1. raiz;
2. no;
3. folha;
4. altura;
5. profundidade.

## Organizacao

- `exemplos-java`: codigo executavel com uma arvore pequena montada em etapas.
- `exemplos-java/exemplo-2-arvore-base`: base didatica mais cadenciada para evoluir para outros tipos de arvore.
- `exemplos-java/exemplo-3-arvore-binaria-abb`: base didatica de arvore binaria de busca com busca eficiente.
- `explicacoes`: texto de apoio com a interpretacao teorica dos conceitos.

## Arquivos principais

- `exemplos-java/ArvoreExemploSimples.java`
- `exemplos-java/exemplo-2-arvore-base/ArvoreBaseDidatica.java`
- `exemplos-java/exemplo-3-arvore-binaria-abb/ArvoreBinariaABBDidatica.java`
- `explicacoes/ARVORES_EXPLICACAO.md`

## Como estudar

1. veja como a raiz e criada primeiro;
2. acompanhe a montagem da arvore em pedaços;
3. identifique quais nos possuem filhos e quais sao folhas;
4. compare altura e profundidade com o desenho mostrado no exemplo.
5. use a base do exemplo 2 para entender como a classe pode crescer ate virar arvore binaria.
6. use o exemplo 3 para ver a busca em ABB e perceber a economia de comparacoes.
